Provides classes to create your own custom widget plugins for
*Qt Widgets Designer* and classes to access *Qt Widgets Designer* components.

In addition, the :class:`~PySide6.QtDesigner.QFormBuilder` class
provides the possibility of constructing user interfaces from UI files at
run-time.

.. note:: This documentation covers the tool *Qt Widgets Designer* and classes
          related to building graphical user interfaces for  :mod:`PySide6.QtWidgets`.
          `Qt Design Studio`_ is a UI composition tool that covers all steps
          of UI design and implementation for :mod:`PySide6.QtQuick` user interfaces.
